iConsolePro(智能健身App)

简介：

  这是一款支持多人在线的健身App，可以连接跑步机、划船机、动感单车等多种实体设备，通过FTMS蓝牙协议解析设备数据，
  
实现数据实时透传，并计算各项运动指标完成 UI 可视化展示，拥有角色切换系统、运动数据云端存储等核心模块，

打造了一个人们在健身时3D虚拟人物实时渲染达到与健身设备同步运动的效果。
